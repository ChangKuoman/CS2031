Descargar cowsay (unix):

	sudo apt-get install -y cowsay

Ver comandos:

	man cowsay

Comandos:

	--

	cowsay hello utec
	
	=> Sale el texto hello utec

	--

	cowsay -e ^^ hello utec

	=> Cambia los ojitos por ^^
	
	--

	cowsay -f tux Tux is Linux\'s mascot!
	
	=> Cambia la vaca a el pinguino de Linux
	=> Caracteres especiales en Linux: '!$#()[]%&
	=> Escaping \'
	=> Quotear '()[]%&'

	--
